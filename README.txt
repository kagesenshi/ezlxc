Installing
===========

::

  git clone https://kagesenshi@github.com/kagesenshi/ezlxc.git ezlxc
  python bootstrap.py
  ./bin/buildout 

Running the command 
====================

::

  ./bin/ezlxc-admin -h

Subcommands
============

create
--------

Create a lxc container

start / stop
------------

Start / Stop a container

list
----

list all containers
